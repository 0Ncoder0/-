# 计算机系统概述

## 计算机发展历史

- 电子计算机的四代演进
  - 第一代（1946-1957）：电子管计算机
  - 第二代（1958-1964）：晶体管计算机
  - 第三代（1965-1974）：集成电路计算机
  - 第四代（1975-至今）：大规模集成电路计算机
- 摩尔定律
  - 集成电路芯片上的晶体管数量每 18-24 个月翻一番
  - 性能提升与功耗限制
- 当代计算机特点
  - 高性能与多核化
  - 低功耗与绿色计算
  - 智能化与专用化

- CPU 时钟周期
  - 时钟频率与周期的关系
